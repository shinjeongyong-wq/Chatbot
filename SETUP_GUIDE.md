### 2단계: Google Drive API 설정

#### A. Google Cloud Console 프로젝트 생성
1. https://console.cloud.google.com/ 접속
2. 새 프로젝트 생성 또는 기존 프로젝트 선택
3. 프로젝트 이름: 예) "hospital-chatbot"

#### B. Google Drive API 활성화
1. "API 및 서비스" > "라이브러리" 이동
2. "Google Drive API" 검색 후 활성화

#### C. API 키 생성 (간단한 방법)
1. "API 및 서비스" > "사용자 인증 정보"
2. "+ 사용자 인증 정보 만들기" > "API 키" 선택
3. 생성된 API 키 복사

**또는**

#### D. OAuth 2.0 클라이언트 ID (더 안전한 방법)
1. "API 및 서비스" > "사용자 인증 정보"
2. "+ 사용자 인증 정보 만들기" > "OAuth 클라이언트 ID"
3. 애플리케이션 유형: "웹 애플리케이션"
4. 승인된 JavaScript 원본: `http://localhost` 추가
5. 클라이언트 ID 복사

---

### 3단계: Gemini API 설정

1. https://aistudio.google.com/app/apikey 접속
2. "Create API Key" 클릭
3. 프로젝트 선택 (위에서 만든 프로젝트)
4. API 키 복사

---

### 4단계: 파일 공유 설정

Google Drive 파일을 다음과 같이 설정:
1. 파일 우클릭 > "공유" 클릭
2. "링크가 있는 모든 사용자" 또는 "조직 내 모든 사용자"로 설정
3. 권한: "뷰어" (읽기 전용)
